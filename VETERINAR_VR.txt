STORYLINE
Scene 01 : The Virtual Greeting
The user spawns at the entrance of a high-tech farm. A friendly, professional Avatar (The Guide) appears beside the user. Before proceeding, the user must select their preferred language—Bahasa Melayu or English—for the audio narration and interface. The Avatar then welcomes the user to the "Smart Ruminant Breeding Lab," explaining that they will learn how to master ruminant breeding techniques using interactive simulations that follow standard Malaysian veterinary practices. The Avatar asks a simple introductory question about breeding to test the user's readiness before prompting them to click a "Proceed" button to enter the farm.

Scene 02 : Herd Behavior Observation
The user is transported into the middle of a paddock. The goal is to observe and identify different cow behaviors. The user sees a variety of cows: some are grazing peacefully, some are just standing/idling, and most importantly, one cow is attempting to mount another. The Avatar explains that these behaviors are "signs of heat" (estrus) and that a smart farmer must recognize them.

The user is transported into a paddock to identify a cow ready for insemination. The goal is to observe and correctly identify "signs of heat" (estrus). The user must spot specific behaviors, including Standing Heat, Mounting Behaviour, and Restlessness. The Avatar provides guidance on why these behaviors are critical markers for the breeding window

ARTIST IMPRESSIONScene 02

The user moves to a specialized Dairy Cow (Lembu Susu).  Before scanning, the user must perform a physical inspection for clinical signs: Lendir Serviks (cervical mucus) and Vulva Swelling. The user notices a "Smart Collar" on the cow’s neck. The user is prompted to "Scan" the cow. A holographic UI appears showing the cow’s internal data: a body temperature of 38.5°C and follicle development status. The user must analyze the data—specifically identifying that the cow is "On Heat"—and note that the ovulation window is within the next 24 hours. The user must analyze this information through an Interactive Decision Tree to confirm the cow is ready for the 24-hour ovulation window. The user must then answer a multiple-choice question to confirm if this cow is ready for breeding.

ARTIST IMPRESSIONScene 04

The user jumps to a digital interface room. On the left side of the screen, they see images/data of cows that have already been successfully bred. On the right side, they see cows that are not yet bred or are unsuitable. The user must answer a series of validation questions based on the images shown (e.g., "Which cow shows optimal genetic traits?"). 
If mistakes were made in Scene 4, the system uses Error-based learning to show Real-time corrections. A Data Analytics Dashboard displays the user's performance metrics: Ketepatan (accuracy), Masa Pelaksanaan (execution time), and Error Rate. The user answers final validation questions to confirm their understanding of genetic record management. This scene emphasizes the importance of digital record-keeping.
