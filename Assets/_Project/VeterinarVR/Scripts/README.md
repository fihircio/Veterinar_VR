This folder owns project code for the Veterinar VR experience.

Suggested split:

- `Core/` for app flow, session state, score management, and scene loading.
- `Gameplay/` for cow interactions, scanning, and AI procedure logic.
- `UI/` for world-space UI controllers and result views.
- `Data/` for ScriptableObject definitions and content contracts.
- `XR/` for project-specific XR wrappers and setup helpers.
